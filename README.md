## Installation

```bash
$ npm install -g cricli
```

## Usage

```sh
$ cricli <options>
```

##### Options
- Commentary
	```bash
	-c, --commentary
	```
- Live
	```bash
	-l, --live
	```
- Scorecard
	```bash
	-s, --scorecard
	```
- Umpires
	```bash
	-u, --umpires
	```
- Recent
	```bash
	-r, --recent
	```
- Toss
	```bash
	-t, --toss
	```
- All
	```bash
	-a, --all
	```

### Prerequisites 
Before running this locally you must have these installed

+ **Node.js**
+ **npm**

### Installing
Installing is quite straight forward. Once you have cloned this repo simply run.

	yarn install
	
After this just run the app using *node*

	node index.js -a
